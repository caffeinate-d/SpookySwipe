# SpookySwipe

A mobile-friendly horror swipe app for GitHub Pages.

## Features
- Big swipe cards
- Apple-like dark/red UI inspired by futuristic horror branding
- Original movie and TV horror database
- Steam horror games mode
- Xbox horror games mode
- Left swipe to pass
- Right swipe to save and open Google search
- Trailer/gameplay trailer button
- Rating/top-picks sort button
- Trending horror button
- Pick For Me button
- Genre filters
- Sort button with up to 100 horror movies/shows/games
- Favorites saved locally in the browser
- Safari and iPhone friendly
